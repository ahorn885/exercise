2. Discovered the project has **no Vercel-Neon Marketplace integration**

   no Neon preview branch existed.

4. Ran `python -m etl.layer0.run --version-tag 1.3`. ETL crashed in
   Phase 2 on `discipline_substitutes` due to a
   `UniqueViolation`.
5. Root-caused: the `Sports_Framework_v10.xlsx` source intentionally
   encodes sub-variants of the same substitute discipline by qualifying
   `substitute_name` while keeping the same `substitute_id`. The schema
   constraint `UNIQUE (target_id, substitute_id, etl_version)` rejected
   these as duplicates. Two known cases:
   - `(D-023, D-001)` Downhill Mountain Running → Trail Running:
     "(sustained downhill)" fidelity 0.85 vs "(rolling)" fidelity 0.30
   - `(D-008b, D-007)` Whitewater Kayaking → Packrafting:
     "(whitewater)" fidelity 0.6 vs "(flat-water)" fidelity 0.4
6. Widened the constraint to include `substitute_name`. Migrated the
   dev branch DB and updated `etl/layer0/schema.sql`. Re-ran the ETL —
   completed end-to-end across all three phases. Report at
   `etl/reports/run-1.3-20260508-161240.md`.

## Reproducing dev setup on Windows (gotchas)

1. **PowerShell execution policy** blocks `npm.ps1`/`vercel.ps1` shims.
   One-time fix:
   ```powershell
   Set-ExecutionPolicy -Scope CurrentUser RemoteSigned
   ```
2. **`python` vs `py`** — Windows ships a Microsoft Store stub at
   `python.exe` that intercepts the bare command. Use `py -m ...` (the
   official launcher) instead of `python -m ...`.
3. **UTF-8 default encoding** — several files in the ETL call
   `Path.read_text()` without `encoding=`, which on Windows defaults to
   `cp1252` and chokes on UTF-8 sources. Set `PYTHONUTF8=1` for the
   process. Real fix would be specifying `encoding="utf-8"` at the call
   sites. (Spotted at least: `etl/layer0/extractors/vocabulary.py:76`.)
4. **`vercel env pull` is interactive** — pass `--yes` for non-TTY use.
5. **Loading `.env` in PowerShell 5.1** — bash `set -a; . ./.env; set +a`
   doesn't work. The `$env:($var)` indirect form doesn't parse either.
   Use:
   ```powershell
   Get-Content .env | ForEach-Object {
     if ($_ -match '^\s*([^#=]+?)\s*=\s*(.*)\s*$') {
       [Environment]::SetEnvironmentVariable($Matches[1], $Matches[2].Trim('"'), 'Process')
     }
   }
   ```
