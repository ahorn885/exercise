● Duplicates & matching

Duplicate detection
Exact file-content match (SHA-256). Re-uploading the same .FIT — or re-syncing an activity we've already pulled — is skipped automatically.
Plan matching
Imports are matched to a planned workout when the date and sport line up, so completing a session updates the plan.
Sport detection
Read from the .FIT sport field when present; we fall back to the file naming otherwise.
